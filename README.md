# NestFinder

Conventions :

variable names :

	must be camel case for example:
	
	mapsActivityMap

Function naming conventions :

	must contain underscore(_) suffix, for example :

	CreateMap_


Layouts :

	componentized, for example, structure :

	-landing
		-top component
		- bottom component
	-common
		-red button
		-blue button
	-profile
		-listings
			-list item
			-list view
		-profile history
			-history item
			-histroy view
